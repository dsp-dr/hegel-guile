* Purpose

implicit protocol map — derived from reading =hegel-core= and =hegel-rust=
source code, validated by CPRR conjectures against live servers.

We maintain this so we know exactly what contract we're implementing,
even if upstream never writes a spec.

* Handshake (C-008 confirmed)

#+begin_example
Client → Server: b"hegel_handshake_start"  (raw bytes, not CBOR)
Server → Client: b"Hegel/0.7"             (raw bytes, not CBOR)
#+end_example

Both sent as HEGL packets on channel 0, message IDs 1 and reply(1).

* HEGL Packet Format (C-009 confirmed)

#+begin_example
[4B magic 0x4845474C "HEGL"]
[4B CRC32]
[4B channel_id]
[4B message_id]           ; bit 31 = reply flag
[4B payload_length]
[NB payload]
[1B terminator 0x0A]
#+end_example

Source: =hegel-core/src/hegel/protocol/packet.py=, =hegel-rust/src/protocol/packet.rs=

** Reply Envelope (C-010 confirmed)

All replies are CBOR maps:
- Success: ~{"result": value}~
- Error: ~{"error": message, "type": error_type}~

Bare CBOR values in replies are a protocol violation.

* Status Strings (C-013 confirmed)

UPPERCASE, case-sensitive. Matches Python =Status= enum names.

| Status | Meaning |
|--------+---------|
| =VALID= | Test case passed |
| =INVALID= | Test case rejected (tc-assume) |
| =INTERESTING= | Test case failed (property violation) |

* Test Case Lifecycle (C-014 implemented)

Server-driven. The server decides when to start/stop test cases.

#+begin_example
1. Client sends run_test on control channel (ch 0)
2. Server acks with {"result": true}
3. Server sends {"event": "test_case", "channel_id": N}  (loop)
4. Client acks test_case, runs thunk with generate/assume on channel N
5. Client sends mark_complete with status on channel N
6. Server sends {"event": "test_done", "results": {...}}  (end)
#+end_example

* Transport

** Unix socket (current, hegel-core ≤0.2.3)

Server writes socket path to stdout, client connects.

** stdio (hegel-core 0.2.3+, hegel-rust 0.2.4+)

~hegel --stdio~ — same HEGL packets over stdin/stdout.
Eliminates socket-file race condition.
Our P2 bead (hegel-guile-m9u).
